Для того, чтобы начать работу:
1) Качаем JDK, если его ещё нет - http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html
2) В переменные среды добавляем JAVA_PATH = { путь к JDK директории }
3) Качаем Eclipse - http://www.eclipse.org/downloads/
4) В еклипсе Help > Eclipse Marketplace > в строке поиска m2e > Maven integration for eclipse (слово с слово)
5) Как и в п. 4 устанавливаем EGit плагин ИЛИ качаем отдельно клиент (например http://code.google.com/p/tortoisegit/wiki/Download?tm=2), клонируем репозиторий и переходим к п. 8
6) Window > Preferences > Team > Git > Поле Clonning repositories - путь к Eclipse Workspace
7) Рестарт еклипса, Window > Show view > Git > Git repositories, в открывшемся окне:
    Clone a git repository > URI = https://github.com/templatevk/GalaxyEclipse.git > Next до конца 
8) В еклипсе File > Import > Existing maven projects и выбираем папку проекта
